### 📄 PolyPDF

## Aplicação em Python com interface simples para traduzir arquivos PDF, detectando automaticamente o idioma e permitindo exportar um novo PDF traduzido.

# 🚀 Funcionalidades

- Abertura de arquivos PDF
- Extração automática de texto
- Detecção automática do idioma original
- Tradução para múltiplos idiomas
- Visualização do texto original e traduzido
- Exportação de PDF traduzido
- Tentativa de preservação do layout original

## ⚙️ Requisitos

- Python 3.8+

## Instalar dependências:

Caso queira testar a versão compilada, basta baixar e iniciar o .exe na pasta .dist/, seja feliz.

- pip install pymupdf googletrans==4.0.0-rc1 langdetect
- ▶️ Como usar
- python main.py
- Clique em Abrir PDF
- Escolha o idioma de destino
- Clique em Traduzir
- Clique em Salvar PDF Traduzido

## ⚠️ Limitações

- PDFs escaneados (imagem) não são suportados sem OCR
- Layout pode não ficar 100% fiel (limitação do formato PDF)
- Tradução depende de serviço externo (pode falhar ocasionalmente)

### 📄 PolyPDF (English)

## A simple Python application with a minimal interface to translate PDF files, automatically detecting the source language and exporting a translated PDF.

# 🚀 Features

- Open PDF files
- Automatic text extraction
- Automatic source language detection
- Translation to multiple languages
- View original and translated text
- Export translated PDF
- Attempts to preserve original layout

## ⚙️ Requirements

- Python 3.8+

## Install dependencies:

If you want to test the compiled version, simply download and run the .exe file in the .dist/ folder, and enjoy.

- pip install pymupdf googletrans==4.0.0-rc1 langdetect
- ▶️ How to use
- python main.py
- Click Open PDF
- Select target language
- Click Translate
- Click Save Translated PDF

## ⚠️ Limitations

- Scanned PDFs (images) require OCR
- Layout may not be perfectly preserved (PDF limitation)
- Translation depends on external service (may fail sometimes)
